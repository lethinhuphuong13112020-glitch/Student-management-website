# Student Management System

## Introduction
This is a basic Student Management Website project developed using PHP, HTML, CSS, JavaScript, and MySQL.

## Features
- Student information management
- Add, edit, delete student data
- Search student information
- Login and logout system
- Class management
- Display student scores

## Technologies Used
- PHP
- HTML
- CSS
- JavaScript
- MySQL

## Project Structure
student-management-system/

│── delete_student.php

Import the `database.sql` file into MySQL to create the database.
